Function handling¶

Anonymous function¶

class pydem.AnonymousFunction¶
get_dy(x, dx)¶
x : array_like
Nominal input configuration to evaluate
dx : array_like
Symmetric variation along each axis based on uncertainty
dy : float
Maximum potential variation based on the dot product of the absolute value of the gradient and the absolute value of the dx vector
get_y(x)¶
x : array_like
point (or list of points) to evaluate based on nominal input configurations
y : array_like
evaluates objective function to perform mapping for the associated dimension in output space process must be repeated over multiple AnonymousFunction objects to accomplish full mapping
